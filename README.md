# 123Fakturera Mini App

This is a small full-stack demo built for the 123Fakturera SOW task.  
It includes a login screen, terms page, and a simple pricelist â€” all connected to a PostgreSQL database.  
Everything runs inside Docker, with a Node.js backend and a React (Vite) frontend.

### Tech Stack
- React (Vite) + Vanilla CSS  
- Node.js (Express)  
- PostgreSQL  
- Docker + Nginx  

### How to Run
1. Clone the repo  
2. Run `docker-compose up --build`  
3. Open [http://localhost](http://localhost) in your browser
